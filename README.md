# Simple Interest Calculator 
 
A simple interest calculator script that calculates simple interest based on user input. 
 
## Description 
 
This script takes the following inputs: 
- Principal - the initial amount of money 
- Rate of Interest - the annual interest rate 
- Time Period - the duration in years 
 
It calculates simple interest using the formula: 
Simple Interest = (Principal x Rate x Time) / 100
